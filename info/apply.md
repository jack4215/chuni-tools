# 賽事申請說明

## 一、申請方式

請透過 [Email](mailto:admin@tsaibee.org) 來信申請，信件內容請包含
```Markdown
主辦人：

賽事名稱：

開放時間：

賽事說明：(非必要)

指定組曲：(Ex. Hydra,MAS....最多三首)
```
## 二、注意事項

1. 同一時間只開放 10 場賽事進行，若您選定的開放日期已達 10 場賽事，將不會受理您的申請，敬請見諒。

2. 賽事開放時間原則以一個月為限，若有特殊需求將另行評估。

3. 本平台顯示時間使用 GMT+8 時區。

3. 我們會盡力確保維持網站功能正常運作，但我們不會明示或隱含保證該等資料均為準確無誤。倘若發生爭議時，本平台不會對任何錯誤或遺漏承擔責任。

4. 本平台保留賽事上架與否的權利。
