# Event Application

## I. Application Method

Please apply via [email](mailto:admin@tsaibee.org). Include the following information:
```Markdown
Organizer:

Event Name:

Opening Date and Time:

Event Description: (optional)

Designated Setlist: (e.g. Hydra, MAS... up to three songs)
```
## II. Important Notes

1. Only ten events can be held simultaneously. If the date you choose already has ten events scheduled, your application will not be accepted. We appreciate your understanding.

2. The event will generally be available for one month. Special requests will be evaluated separately.

3. The time displayed on this platform is in GMT+8 time zone.

4. We strive to ensure the website functions properly, but we do not guarantee the accuracy of the information provided, either explicitly or implicitly. In case of any disputes, this platform will not be held liable for any errors or omissions.

5. The platform reserves the right to approve or reject event listings.
